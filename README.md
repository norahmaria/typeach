#### 🚀 Start by installing the dependencies with `yarn`

#### Storybook

```ts
// run storybook for dev
yarn story

// build storybook
yarn story:build
```

#### Testing

```ts
// run tests
yarn test

// update snapshots
yarn test:update
```

#### Cypress

```ts
// open cypress
yarn cy:open

// open component testing
yarn cy:ct
```

#### Build the project for prod / publish with `yarn build`
